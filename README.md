# Engram

Engram is a modal-based study editor and Anki flashcard factory.

## Run locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Start the app:
   `npm run dev`

## Optional environment variables

- `API_KEY`: Enables generation features.
